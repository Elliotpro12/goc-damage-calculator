# Guardians of Cloudia — Calculadora de Daño v0.7

Calculadora de daño táctica para Guardians of Cloudia.

**Desarrollado por:** xDarKz  
**UI / React:** Claude (Anthropic)

## Funcionalidades
- Cálculo de daño principal (físico/mágico, crítico, debuff, PvP, vs clase)
- Heavy ATK — probabilidad de acierto y efecto DMG recibido
- Ataque Cargado con stacks y Charged Burst
- Partners: Olivia (❄️ daño), Isabel (🌿 curación Skill 1)
- Soporte ES / EN
- Adaptado para PC y móvil

## Uso local
```bash
npm install
npm run dev
```
